## 오픈소스SW개론 2분반 테스트 (실습)

1.	top: 시스템의 실시간 프로세스와 자원 사용 현황을 보여줌
·	CPU, 메모리 사용량 확인 가능!
·	종료: q

2.	ps: 현재 실행 중인 프로세스 목록 출력함
·	주요 옵션:
  - ps -e : 모든 프로세스 표시
  - ps aux : 상세 정보 표시

3.	jobs: 현재 셸에서 실행 중인 작업 목록 확인함
·	백그라운드 작업 관리에 유용
·	예시: jobs -l

4.	kill:	프로세스 종료 명령함
·	사용법: kill [PID]
·	강제 종료: kill -9 [PID]
